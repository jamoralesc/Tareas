Se utilizó else if para mostrar los rangos de meses a validar
para identificar la estación del año.

En vista que es un programa sencillo, sin ocuparnos en detalles de mayor
complejidad o escenarios que llegaran a causar un mal funcionamiento y 
considerando que de momento es obvio que solo se deberán ingresar 
números del 1 al 12, se dejó en el else como interpretación cuando se
registre un número 11, 12 o 1. En este caso se mostrará invierno.

Para el resto de estaciones, si se encuentra definido un rango específico
de meses.
